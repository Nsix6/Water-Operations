# MVP Scope

# Included Features

## Customer Features

- Registration
- Login
- Profile management
- Order placement
- Order history
- Email notifications

---

## Driver Features

- Driver login
- Assigned delivery view
- Delivery status updates
- OTP confirmation

---

## Administrative Features

- User management
- Order management
- Driver assignment
- Delivery monitoring
- Operational dashboards
- Audit visibility

---

# Included Infrastructure

- Web application
- Backend API
- PostgreSQL database
- Email notification system
- Authentication system

---

# Excluded Features

The following features are excluded from the initial MVP:

- Native mobile applications
- Loyalty systems
- Advanced analytics
- Multi-provider marketplace
- Route optimization
- Inventory forecasting
- AI functionality
- Real-time GPS tracking
- Automated dispatch optimization

---

# MVP Objective

The objective of the MVP is to validate operational workflows, reduce revenue leakage, improve delivery accountability, and centralize customer ordering operations.
